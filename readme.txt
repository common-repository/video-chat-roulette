== Description ==
With chat roulette you can let visitors of your website to stay more. Chat roulette is based on Flash and would work in any desktop browser supporting Flash Player.
Application tries to establish peer-2-peer connection with opponent and in case it is failed there will be connection with central streaming server.

Concept of this application is very good to make new friends and dating

== Installation ==

1) Download and install plugin
2) Activate the plugin at Wordpress admin panel -> Plugins
3) Follow instruction provided on the plugin menu
